# Códigos Maliciosos
Laboratório: Construção de códigos maliciosos

O laboratório teve como objetivo o estudo de códigos maliciosos construidos utilizando-se a linguagem de programação python. Apesar de serem códigos que podem causar danos a vítimas reais, os scripts trazem nada mais do que funções básicas da linguagem de programação Python, fato que nos alerta sobre a possibilidade de utilizar conhecimentos benéficos para o mal.
